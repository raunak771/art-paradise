How to Join on Bedrock Edition

  1. Open Minecraft on your device.
  2. Click on "Play".
  3. Navigate to the "Servers" tab.
  4. Scroll down and click "Add Server".
  5. Enter the server name: Art Paradise
  6. Enter the server address: mc.artparadise.uk
  7. Set the port to: 19132
  8. Click "Save" and then select the server to join.

  5. Enter the server address: mc.artparadise.uk
